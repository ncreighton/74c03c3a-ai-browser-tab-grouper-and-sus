# AI Browser Tab Grouper and Suspender Extension

Analyzes open tabs to suggest groups and identify memory-heavy pages that could be suspended. Classifies pages into categories like Work, Development, Social, and Entertainment based on content analysis.

## Key Features

- Classifies tabs into 8 groups: Work, Development, Social, News, Shopping, Entertainment, Email, Finance
- Identifies memory-heavy pages (many scripts, iframes, or media elements)
- Flags pages that are candidates for suspension to save memory
- Shows confidence scores for group assignments
- Analyzes page title, meta tags, URL, and content for classification

## How It Works

Click the extension icon on any tab. The extension scans the page content, URL, and metadata to determine which category the tab belongs to. It also counts resource-heavy elements. All analysis is local.

## What You Get

- Instant tab classification with group suggestions
- Memory usage indicators for each tab
- Works on any webpage
- Privacy-first — all processing is local

## Installation

### Chrome
1. Download `chrome_extension.zip`
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download `firefox_extension.zip`
2. Go to `about:debugging`
3. Click "Load Temporary Add-on"
